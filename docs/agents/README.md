# Agent Roster

CivicPulse uses five role-based agents. @pm coordinates all sprint work and assigns every task to exactly one agent. No task is ownerless.

| Handle | Role | Doc |
|--------|------|-----|
| @pm | Project Manager — sprint planning, task assignment, DoD gating | [pm-agent.md](pm-agent.md) |
| @backend | Backend Developer — ASP.NET Core API, EF Core, external clients, scoring logic | [backend-agent.md](backend-agent.md) |
| @frontend | Frontend Developer — Blazor Server UI, components, charts | [frontend-agent.md](frontend-agent.md) |
| @infrastructure | DevOps / Infra — Docker, CI/CD, DB container | [infrastructure-agent.md](infrastructure-agent.md) |
| @qa | QA / Test Engineer — xUnit unit + integration tests | [qa-agent.md](qa-agent.md) |
